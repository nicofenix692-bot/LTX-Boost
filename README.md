# :rocket: LTX Boost

**LTX Boost** es un optimizador avanzado para **Windows 11**, diseñado para mejorar el rendimiento, limpiar archivos innecesarios y potenciar tu experiencia de **gaming competitivo**.

---

## :sparkles: Características principales
- :broom: **Limpieza profunda** de archivos temporales y basura del sistema.  
- :video_game: **Optimización para gaming**, aumentando FPS y estabilidad.  
- :gear: **Ajustes personalizables** para rendimiento según tus necesidades.  
- :desktop: **Interfaz sencilla y eficiente**, lista para usar.

---

## :tools: Tecnologías utilizadas
- **C# / .NET 9**  
- **WPF** para interfaz gráfica  
- **PowerShell** para scripts de mantenimiento y limpieza
