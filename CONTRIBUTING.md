# Contribuir a LXT Boost

¡Gracias por considerar contribuir a LTX Boost! Apreciamos tu ayuda para mejorar el proyecto. Sigue estos pasos para comenzar:

## Cómo Contribuir

1. **Haz un Fork del Repositorio: Haz clic en el botón “Fork” en la esquina superior derecha de este repositorio para crear tu propia copia..
2. **Realiza tus Cambios: Clona tu fork de forma local y crea una nueva rama para tus cambios.
3. **Haz commit de tus cambios en la rama `testing` Asegúrate de que los mensajes de commit sean claros y descriptivos.
4. **Cuando tus cambios estén listos, envía un pull request al repositorio principal.

## Lineamientos

- **Estilo de Código: Asegúrate de que tu código siga el estilo de codificación del proyecto. Un estilo consistente ayuda a mantener la legibilidad y la calidad.
- **Mensajes de Commit: Escribe mensajes de commit claros y concisos. Un buen mensaje debe describir qué cambió y por qué.
- **Documentación: Incluye documentación para cualquier nueva función o cambio. Esto ayuda a otros a entender tus contribuciones y a usar las nuevas funciones.
- **Pruebas: Prueba tus cambios cuidadosamente antes de enviar un pull request. Esto ayuda a detectar problemas a tiempo y garantiza la estabilidad.

## Obtener Ayuda

Si necesitas ayuda o tienes preguntas, no dudes en abrir un issue o iniciar una discusión. Estamos aquí para ayudarte.

¡Gracias por tus contribuciones y por ayudar a que LTX Boost sea mejor!
